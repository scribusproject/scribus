Créer et manipuler les styles

createCharStyle(...)
createCharStyle(...)

Crée un style de caractère. Cette fonction accepte les paramètres nommés suivants:

  • "name" [requis] -> nom du style de caractère à créer (premier paramètre)
  • "font" [optionnel] -> nom de la police de caractère à utiliser (second paramètre)
  • fontsize [optionnel] -> taille de la police de caractère (3ème paramètre - nombre réel)
  • "features" [optionnel] -> détails typographiques définis sous la forme d'une liste d'éléments séparés par des virgules (sans espaces!), pris parmis les éléments suivants (4ème paramètre - chaîne de caractère):
    • inherit
    • bold
    • italic
    • underline
    • underlinewords
    • strike
    • superscript
    • subscript
    • outline
    • shadowed
    • allcaps
    • smallcaps
  • "fillcolor" [optionnel], fillshade [optionnel] -> définie la couleur et la teinte de fond (5ème paramètre: chaîne, 6ème paramètre: nombre réel 1.0 = 100%)
  • "strokecolor" [optionnel], strokeshade [optionnel] -> définie la couleur et la teinte de filet (7ème paramètre: chaîne, 8ème paramètre: nombre réel 1.0 = 100%)
  • baselineoffset [optionnel] -> décalage par rapport à la ligne de base(9ème paramètre)
  • shadowxoffset [optionnel], shadowyoffset [optionnel] -> décalages de l'ombre portée si celle-ci est utilisée(10ème et 11ème paramètres)
  • outlinewidth [optionnel] -> largeur du filet si celui-ci est utilisé (12ème paramètre)
  • underlineoffset [optionnel], underlinewidth [optionnel] -> options de soulignement si utilisé (13ème et 14ème paramètres)
  • strikethruoffset [optionnel], strikethruwidth [optionnel] -> options de barré si utilisé (15ème et 16ème paramètres)
  • scaleh [optionnel], scalev [optionnel] -> échelle horizontale et verticale des caractères (17ème et 18ème paramètres - nombres réels, 1.0 = 100%)
  • tracking [optionnel] -> valeur d'approche du texte(19ème paramètre - nombre réel, e.g. -50 = -5.0%)
  • "language" [optionnel] -> code de language (20ème paramètre - chaîne de caractère: en = English)
  • "fontfeatures" [optionnel] -> chaîne de caractère contenant une liste de fonctionnalités OpenType séparées par des virgules (21ème paramètre):
    • Ligatures:
      • -liga: désactive les ligatures commmunes
      • -clig: désactive les ligatures contextuelles
      • +dlig: active les ligatures discrétionnaires
      • +hlig: active les ligatures historiques
    • Position des caractères:
      • +subs: active les indices
      • +sups: active les exposants
      • +ordn: active les ordinaux
    • Capitales:
      • +smcp: active les petites capitales
      • +c2sc: active les capitales en petites capitales
      • +pcap: active les petite caps
      • +c2pc: active les capitales en les petite caps
      • +unic: active l'écriture unicamérale
      • +titl: active le titrage
    • Chiffres:
      • +lnum: active les chiffres alignés
      • +tnum: active les chiffres en style ancien
    • Largeur des chiffres:
      • +pnum: active la largeur de chiffre proportionnelle
      • +tnum: active la largeur de chiffre tabulaire
    • Fractions numériques:
      • +frac: active les fractions diagonales
      • +afrc: active les fractions empilées
    • Zéro numérique:
      • +zero: active le zéro barré
    • Jeux de styles:
      • +ss01: active le 1er jeu de style
      • ...
      • +ss20: active le 20ème jeu de style

Tout atttribut de style non explicitement définie verra sa valeur héritée du style de caractère par défaut.

Du fait du nombre important d'arguments de cette fonction, il est fortement conseillé d'utiliser la syntaxe par mots clés lors de son appel. Par example si vous souhaitez modifier uniquement la taille de police, n'appelez pas la fonction de la façon suivante:

  • newStyle = createCharStyle("Nouveau Style", "Arial Regular", 12)

mais utilisez plutôt cette syntaxe à la place:

  • newStyle = createCharStyle("Nouveau Style", fontsize=12)

Pour plus de détails à ce sujet et notamment sur la façon de définir l'approche, veuillez consulter la page wiki Text and Text Manipulation.

createCustomLineStyle(...)
createCustomLineStyle(nomStyle, style)

Crée le style de filet personnalisé 'nomStyle'.

Cette fonction accepte une liste de dictonnaires en tant que paramètre du style. Chaque dictionnaire représente un sous-filet du style et accepte les clés suivantes:

  • Color [optionnel] -> nom de la couleur à utiliser (chaîne)
  • Dash [optionnel] -> type de ligne à utiliser (entier)
  • LineEnd [optionnel] -> type de fin de filet à utiliser (entier)
  • LineJoin [optionnel] -> type de jointure de filet à utiliser (entier)
  • Shade [optionnel] -> teinte de couleur (entier)
  • Width [optionnel] -> largeur de filet (nombre réel entier)

createParagraphStyle(...)
createParagraphStyle(...)

Crée un style de paragraphe. Cette fonction accepte les paramètres nommés suivants:

  • "name" [requis] -> spécifie le nom de style de paragraphe
  • linespacingmode [optionnel] -> specifie le mode d'interlignage. Les valeurs possibles sont:
    • interlignage fixe: 0
    • interlignage automatique: 1
    • aligner le texte sur la grille: 2
  • linespacing [optionnel] -> spécifie la valeur d'interligne si l'interlignage fixe est utilisé
  • alignment [optionnel] -> spécifie l'alignement des paragraphes
    • gauche: 0
    • centre: 1
    • droit: 2
    • justifié: 3
    • étendu: 4
  • leftmargin [optionnel], rightmargin [optionnel] -> spécifie les marges gauches et droites
  • gapbefore [optionnel], gapafter [optionnel] -> spécifie l'espacement avec le paragraphe précédent et le paragraphe suivant
  • firstindent [optionnel] -> le décalage de la première ligne
  • hasdropcap [optionnel] -> spécifie si les lettrines sont activées (1 = oui, 0 = non)
  • dropcaplines [optionnel] -> la hauteur des lettrines en nombre de lignes si l'utilisation de lettrines est activée
  • dropcapoffset [optionnel] -> décalage horizontal des lettrines si l'utilisation de lettrines est activée
  • "charstyle" [optionnel] -> nom du style de caractère à utiliser
  • "bullet" [optionnel] -> chaîne de caractère à utiliser pour les puces
  • "tabs" [optional] -> a list of tab definitions
    • un tab est défini par un triplet de valeur sous la forme suivante (position, type, fillchar)
    • position [requis] -> nombre réel représentant la position
    • type [optionnel] -> gauche: 0 [default], droit: 1, point: 2, virgule: 3, centré: 4
    • fillchar [optionnel] -> le caractère utilisé pour remplir l'espace occupé par le TAB, aucun par défaut.

Si vous désirez omettre un certain nombre d'éléments, cette fonction n'acceptera pas de valeurs null, i.e., une série de virgule. Vous devez spécifier un paramètre entier pour chacun des paramètres omis. Par example, si vous désirez spécifier uniquement un nom de style de paragraphe et un nom de style de caractère, votre appel de fonction ressemblera à ceci:

  • scribus.createParagraphStyle("MyNewStyle",0,0,0,0,0,0,0,0,0,"MyCharStyle")

Cependant, si vous désirez spécifier uniquement un nom de style et un mode d'interlignage, vous pouvez alors limiter le nombre d'argument passé dans votre appel de fonction:

  • scribus.createParagraphStyle("MyOtherNewStyle", 1)
